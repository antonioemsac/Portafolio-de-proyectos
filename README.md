Página web de presentación profesional con ocho proyectos de inteligencia territorial, análisis de datos y desarrollo tecnológico aplicados al Istmo de Tehuantepec y el Corredor Interoceánico (CIIT). Incluye herramientas interactivas de competitividad municipal, mapas de riesgos y capacidades productivas, simuladores de localización estratégica y observatorio geotérmico, todos accesibles directamente desde la plataforma GitHub Pages del autor.
